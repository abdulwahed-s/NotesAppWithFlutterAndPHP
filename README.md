# Notes App (Flutter Frontend + PHP Backend)

This is a feature-rich **Notes App** built using **Flutter** for the frontend and **PHP with MySQL** for the backend. The app allows users to create, read, update, and delete notes seamlessly, providing a smooth and intuitive user experience.

---

## Features

- **User-Friendly Interface**: Built with Flutter for a responsive and modern UI.
- **Secure Backend**: PHP and MySQL ensure robust and reliable data storage.
- **CRUD Operations**: Full functionality to create, read, update, and delete notes.
- **Scalable Architecture**: Designed to be easily extended for future enhancements.
- **Cross-Platform**: Runs smoothly on both Android and iOS devices.

---

## Tech Stack

- **Frontend**: Flutter & Dart  
- **Backend**: PHP  
- **Database**: MySQL  
- **Hosting**: Local Hosting

---

## Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/abdulwahed-s/NotesAppWithFlutterAndPHP.git
cd NotesAppWithFlutterAndPHP
```

### 2. Set Up the Backend
1. Upload the PHP files to your server.
2. Configure the database connection in `connect.php`.
3. Import the SQL file (`database.sql`) to create the required database structure.

### 3. Set Up the Frontend
1. Open the project in your Flutter development environment.
2. Update the API URLs in the code to point to your backend server.
3. Run the app using:
   ```bash
   flutter run
   ```
